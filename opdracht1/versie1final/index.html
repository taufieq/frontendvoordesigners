Opdracht 1 - versie 1

Hier komt de HTML van de interactieve demo te staan voor de eerste opdracht: Ontwerp een user interface voor een gegeven use case. Werk je ontwerp uit in HTML, CSS en Javascript om te kunnen testen in een browser.

  1. Als ik een lijst films heb gevonden, wil ik leuke films kunnen opslaan om ze thuis te ...
  2. In een lijst gevonden studentenhuizen wil ik details kunnen bekijken om te bepalen of de kamer geschikt is.
  3. In een verzameling muzieknummers wil ik de volgorde kunnen veranderen om een vette playlist te maken.
  4. Als ik foto's zoek wil ik kunnen selecteren op kleur om verassende zoekresultaten te krijgen.
